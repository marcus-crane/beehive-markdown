Saunoa le Minista o Tagata Atumotu o le Pasifika, “O le paketi 2008 o lo’o fesoasoani tele mo le soifua manuia mo le lumana’i o tagata pasifika.”

O tatou tagata pasifika ua telē se manuia ua maua i le faigamalo a le Leipa. Ua maualalo le pasene o tagata pasifika e le’o faigaluega (unemployed) - Tesema 1999 sa 12.2% tagata ua leai ni galuega a’o lenei ua 4.7% Tesema 2007, ua fa’atupulaia foi le manuia o tulaga o a’oa’oga ma totogi maua mo tagata pasifika.

O le fa’apaū o lafoga o le a fa’aopopo ai le tupe e maua e matua o lo’o faigaluega, o le a fesoasoani tele i aiga talavou, fa’apea foi aiga matutua aua ua le si’itia foi tau o le tamaoaiga ona o le lē mautu o lea tulaga mai atunu’u i fafo.

Saunoa le Minista Hon Laban “O lenei manuia o le a fa’aopopo ai le tupe maua e tatou tagata pasifika.”

O tagata pasifika o lo’o maualalo totogi po’o i galuega o lo’o fa’aitiitia itula (part-time) o le a manuia i le fesuiaiga o le lafoga. O ē o lo’o totogi i le o’gatotonu seia o’o i le totogi maualuga (middle to high income) o le a manuia foi ona ua si’i le totogi fa’atapula’a mo lafoga (high tax bracket). Mo aiga ta’itasi o lo’o galulue i galuega tumau, mai le aso muamua o Oketopa o le suiga o le lafoga mai le $620 aga’i le $1460 i le tausaga, po’o le $12 aga’i le $28 i le vaiaso.

O se fa’ata’ita’iga o se aiga Pasifika - o le tamā o le aiga e maua le $45,000 i le tausaga a’o le tinā e maua le $25,000 i le tausaga ae to’afa le fanau o le ta’i 8, 10, 14 ma le 16 tausaga le matutua o le fanau, o le a maua le fa’asiliga o le $50 i le vaiaso, po’o le $2590 i le tausaga ma le lafoga o le fesoasoani o matua e faigaluega (Working for Families). A’o o matua matutua (grand parents) o le a iai se fa’asiliga o le $45.88 i le ta’ilua vaiaso.

O tatou matua matutua Pasifika o lo’o maua le SuperGold card o le a leai se totogi o le feoa’i i pasi i itula fa’atagaina po’o (off peak), o le a si’itia foi le tupe fesoasoani mo le faia lea o mea fa’alogo (hearing aids).

Saunoa le Minista Hon Luamanuvao Winnie Laban “E fa’apea foi ona fa’afaigofie nisi o tulaga mo alauni mo ē o lo’o a’oa’oina i univesete po’o aoga maualuluga (tertiary education), fa’aopopo skolasipi (scholarships), atoa ai ma le lagolagoina i mea tau seleni auā le fa’alaua’itele o le silafia mo matāta eseese.”

O le Paketi 2008 ua saunia ai le fa’apaū o lafoga, aemaise foi le fesoasoani tele i matāgaluega eseese a le malo, e pei o falema’i, faleaoga, fa’ataugofie fomai, fa’ataugofie fale a le malo, aemaise foi o polokalame eseese auā le fa’atupulaia o le silafia o galuega eseese.

Saunoa le Minista o Tagata Atumotu o le Pasifika Hon Luamanuvao Winnie Laban “O lenei Paketi 2008 o se Paketi mo le manuia o tatou tagata. O lo’u fiafia ia o lo’o avea a’u ma faipule ma Minista o le malo o le Leipa, ina ia ou galue malosi mo le soifua manuia o tatou tagata mo le lumana’i.
